Chapter 1. Introduction

Chapter 2. Luntbuild Element

All XML responces from Luntbuild have the <luntbuild> element as the document root. The root object contains the <system> element, with all of the system wide information, and the <result> element. The result element contains all the elements returned by the query. The structure of the results depends on the query. The <luntbuild> element may also have a child element named <error> that contains any error messages encountered.

<luntbuild> element:

<luntbuild>
	<system>
		...
	</system>
	<results>
		...
	</results>
	<error>Unsupported method</error>
</luntbuild>

Chapter 10. Notifier Property Element

Child elements in blue only exist in <notifierProperty> element stored in the <system> element.

<notifierProperty> element:

<notifierProperty>
	<key>com.luntsys.luntbuild.notifiers.EmailNotifier#Email address of the sender</key>
	<value>luntbuild@<hostname></value>
	<displayName>Email address of the sender</displayName>
	<description>This property is optional. If specified, Luntbuild will use this email as the sender address when sending out email notifications. Otherwise, the sender address will be "luntbuild@&lt;hostname&gt;", where &lt;hostname&gt; is the host name of the build machine.</description>
	<notifierClass>com.luntsys.luntbuild.notifiers.EmailNotifier</notifierClass>
</notifierProperty>
