Installing or updating Luntclipse

  1. Download Luntclipse 1.3 from Javaforge  or Sourceforge
  2. Unzip contents of the zip file to the Eclipse "plugins" directory and restart Eclipse
  3. Alternatively use Eclipse Help/Software Updates/Find and Install and create new update site with Url: http://luntbuild.javaforge.com/luntclipse-release
  4. Open Luntclipse view by selecting menu item Window > Show View > Other > Luntclipse > Luntbuild

Installing standalone Luntclipse RCP

To install standalone version of Luntclipse (Luntclipse RCP):

  1. Download Luntclipse 1.3 RCP for your platform from Javaforge  or Sourceforge
  2. Unzip contents of the zip file
  3. On some platforms you will have to set env. variable MOZILLA_FIVE_HOME to use SWT Browser. See SWT FAQ for details.
