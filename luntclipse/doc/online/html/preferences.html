Preferences

Currently Luntclipse supports following preferences:

  • Refresh Interval (s): refresh interval in seconds to update schedule, build data from Luntbuild
  • Use Notify Tray (requires restart): if selected, notify tray is used for Luntbuild event notification
  • Run Notify Tray even if View is Hidden (requires restart): if selected, the tray notification runs even if Luntclipse view is hidden
