2006/4/1: Luntbuild 1.3.2 beta is available with new SVN Adaptor  at Javaforge. It contains re-implemented SVN Adaptor based on JavaSvn. Implemented by Alexei Vidmich, thanks Alex. It improves and simplifies SVN Adaptor including ssh access. External SVN installation is no longer needed. I would very much appreciate if you guys who use SVN Adaptor test it and report any problems (do not use luntbuild 1.3.2 beta if you are using LDAP access). I intend to release final version 1.3.2 with new SVN Adaptor in about two weeks.

2006/1/06: Final release of Luntbuild 1.3 is expected around January 15. It will include basic LDAP authentication support and improved create/edit project support for Luntbuild Hessian remote service. Luntclipse plugin (with same version as Luntbuild) will be released at the same time. It will support all the Luntbuild features with exception of admin only features (Properties, Users, Import/Export). Standalone RCP version of Luntclipse will be also available.

2005/03/24: Release plan for Luntbuild 1.2:

  1. Release Luntbuild 1.2 beta1 in next two weeks.
  2. After the release, the DB schema is frozen, and only bug fixes will go to final 1.2 release.
  3. DB migration script from Luntbuild 1.1.1 will be provided for Luntbuild 1.2 beta1.
  4. If everything goes well with Luntbuild 1.2 beta, the final Luntbuild 1.2 should be released sometimes around the end of April.
Please consider the dates tentative 8-).
