Overview

Luntclipse is Eclipse plugin that provides functionality for working with builds. It is possible to watch status of any build or trigger a build in any build schedule. Luntclipse communicates with Luntbuild through Hessian web service protocol and HTTP (log detail). Luntclipse possibilities are limited by range of APIs exported by Luntbuild . Currently Luntclipse supports all functionality offered by Luntbuild with exception of User management, Luntbuild properties management and Import/Export.

Feature List

  • create and manage multiple Luntbuild connections
  • watch build information (similar to Luntbuild home page)
  • trigger build
  • display detail of a build log
  • display detail of a revision log
  • display detail of a system log
  • search, move, delete history builds
  • view history builds
  • Create/edit project
  • browse Luntbuild web application
