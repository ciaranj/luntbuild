Following is a list of examples on how to use Luntbuild Remote API:

  • Edit Properties - This class gives an example of using luntbuild web service API to edit propertie of a project
  • Search Builds - This class demonstrates search builds in the system, and how to access build information
  • Trigger Build  - This class gives an example of using luntbuild web service API to trigger a build
  • LuntbuildConnection - This class demonstrates access to project, schedule and build data. It shows how to create, edit and delete a project (supported since Luntbuild 1.3). This class is part of the Luntclipse, Eclipse plugin for Luntbuild.

Javadoc documentation for Luntbuild Remote API is here.
