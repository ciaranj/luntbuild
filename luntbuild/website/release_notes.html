Luntbuild-1.1.1 Release Notes

  • Changes since version 1.1
    1. Sender's email address can be specified at the Properties configuration page.
    2. Trunk directory can be specified for Subversion VCS adaptor.
    3. Utilized Cvs history command to speed up revision checking.
    4. Fixed the bug that Clearcase VCS adaptor reports error when calling "mkview".
    5. Fixed the bug that post-build target is ignored by Luntbuild.
    6. Fixed the bug that Cvs adaptor log command works in-correctly with some version of Cvs.
    7. Fixed the bug that relative reference to other pages in artifacts directory can not be displayed correctly.

  • Upgrade from version 1.1
    1. Copy all files under <luntbuild_install_dir>/db and save them somewhere else.
    2. Follow installation instructions in readme.txt.
    3. Stop Luntbuild web application, copy back previously saved db files to <luntbuild_install_dir>/db directory and override existing files.
