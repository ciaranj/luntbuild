Web Luntbuild QuickBuild

Luntbuild is based on quite a few open source libaries, frameworks and databases. It leverages great work of many dedicated developers in following projects:

Libraries and Frameworks

  • Tapestry component based web application framework
  • Tacos AJAX based components for Tapestry
  • Spring system for assembling components via configuration files
  • Acegi Spring security services
  • Hibernate ORM database access
  • JavaSVN  Subversion client library
  • IzPack installer generator
  • Ognl Object-Graph Navigation Language used by Tapestry
  • Quartz job scheduling system
  • Xstream library to serialize objects to XML and back again
  • Velocity template engine
  • Jetty Java HTTP Server and Servlet Container
  • Ant Java-based build tool
  • Maven software project management and comprehension tool
  • Rake Ruby Make
  • Ehcache java distributed cache
  • Apache Commons open source reusable Java components
  • Log4j Java logging
  • Hessian binary web service protocol

Databases (free)

  • H2  the free SQL database written in Java
  • Derby a relational database implemented entirely in Java
  • HSQLDB SQL relational database engine written in Java
  • ProgreSQL open source relational database system
  • MySql popular open source database

Host sites

  • Sourceforge  is the world's largest Open Source software development web site
  • Javaforge hosting by Javalobby

Website tools

  • Tigra Hints
  • Tigra Menu
  • Tigra Scroller
