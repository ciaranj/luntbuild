Url to access luntbuild servlet


The servlet url should be specified correctly here in order to construct urls used in the notification message correct. Normally this value should be http://<server>:<port>/luntbuild/app, where <server> stands for your build server name or ip address, and <port> stands for the port number you used to access luntbuild. If this property is left empty, luntbuild will default this value to be "http://<server_ip>:8080/luntbuild/app", where <server_ip> stands for the ip address of the build server.
