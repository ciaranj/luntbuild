Chapter 8. Creating the Project Builder(s)

Click on Builders tab.

Click on New Builder icon in upper right corner of the tab's frame.

Select ant builder.

Modify command to run ant if needed (using -Ddebug=_debug, using -v).

Set build script path (might be build.xml, build.xml ant configuration file in the top level project directory).

Set build target(s), consider using ${build.schedule.name} to use targets based on name of the currently executed schedule variable.

Set build success condition (default is OK).
