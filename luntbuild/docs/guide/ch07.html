Chapter 7. Creating a Project

Click on Project Tab.

Click on New Project icon in upper right corner of the tab's frame.

  1. Name your project, keep in mind that project work directory will be in subdirectory named as your project in Luntbuild main work directory.

  2. Specify project description.

  3. Specify string for next build version, keep in mind the name will be used as subdirectory in the publish directory.

  4. Select project admins (previously created using Users tab).

  5. Select project builders.

  6. Select project viewers.

  7. Select notification methods.

  8. Select notification users.

  9. Select log level.
